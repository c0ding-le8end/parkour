# parking_management

<p>•A website that lets users to book a parking slot available in a street at a given time.<br/>
•Streets that have a plenty of parking spaces available are marked as green<br/>
•Streets that have a limited number of parking spaces are marked as red</br></p>
<p>The application consists of a flutter web-app that sends and recieves data through a REST api that connects to a Flask backend server.
The Flask backend is connected to a mySQL database</p>
